    After-Effects¶

    An effortless post install script for Ubuntu

    Why¶

    Upgrading a Linux machine comes with lot of headache. Somethings might break, somethings might not work or you might just want to start fresh. Beauty of Linux is almost everything is scriptable. Re-installing is just a matter of putting your favorite distribution on USB, install and configuring it. If you do it often and over a fleet of machines, then its a pain. You can run a few scripts to automate it, but they are not so flexible. A package changed its name? PPA not available for the current release? You need to edit your script, test it and run it.

    See Getting Started.
